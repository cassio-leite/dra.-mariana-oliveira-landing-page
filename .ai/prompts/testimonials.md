# Objetivo

Transformar a seção `Testimonials.tsx` em uma área premium de prova social para uma landing page de psicologia clínica.

# Contexto

O projeto utiliza:

- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- BlurFade
- Design minimalista e sofisticado
- Paleta em tons bege e sage
- Bordas arredondadas
- Glassmorphism suave
- Espaçamento generoso
- Animações delicadas

A página deve transmitir:

- Acolhimento
- Confiança
- Serenidade
- Sofisticação
- Credibilidade

Evite aparência de landing pages genéricas ou excessivamente comerciais.

# Objetivo visual

Criar uma seção elegante e emocionalmente acolhedora, inspirada em clínicas premium e sites minimalistas.

# Layout

Criar:

- Badge superior em letras maiúsculas:

"Depoimentos"

- Título principal:

"O cuidado começa pela confiança."

- Subtítulo:

"Relatos de pessoas que encontraram um espaço seguro para compreender emoções, fortalecer a autoestima e construir uma vida mais equilibrada."

Abaixo do título:

Uma grade com 3 cards de depoimentos.

Desktop:

3 colunas.

Tablet:

2 colunas.

Mobile:

1 coluna.

# Cards

Cada card deve possuir:

- Fundo branco com transparência suave.
- Backdrop blur discreto.
- Bordas arredondadas entre 2rem e 2.5rem.
- Sombra elegante e suave.
- Hover muito sutil.
- Padding generoso.
- Boa hierarquia tipográfica.

Adicionar:

- Ícone de aspas decorativo.
- Cinco estrelas discretas.
- Texto do depoimento.
- Nome anonimizado.
- Idade aproximada.

Exemplo:

★★★★★

"Me senti acolhida desde a primeira conversa. A terapia me ajudou a compreender sentimentos que eu não conseguia expressar."

Paciente • 34 anos

★★★★★

"Um ambiente leve, respeitoso e sem julgamentos. Hoje consigo lidar melhor com a ansiedade e as relações do dia a dia."

Paciente • 29 anos

★★★★★

"Aprendi a olhar para mim com mais gentileza. Foi um processo transformador."

Paciente • 41 anos

# Microinterações

Adicionar:

- BlurFade com delay escalonado.
- Hover com translate-y muito suave.
- Transição de 300ms.
- Pequeno aumento da sombra no hover.

As animações devem transmitir calma e sofisticação.

# Responsividade

Garantir excelente experiência em:

- Mobile
- Tablet
- Desktop

# Importante

Não alterar a identidade visual existente.

Manter consistência com:

- Hero
- About
- Process

Priorizar:

- Elegância
- Leveza
- Sofisticação
- Espaço em branco
- Leitura confortável

Gerar o componente completo `Testimonials.tsx`, pronto para uso, utilizando React + TypeScript + Tailwind CSS.
