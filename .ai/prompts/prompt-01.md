# Melhorar Navbar — Etapa 6 (Menu Mobile Premium)

Objetivo:

Substituir o menu mobile atual por um menu premium utilizando o Sheet do shadcn/ui.

Ao abrir:

Adicionar um overlay:

bg-black/20
backdrop-blur-sm

O menu deve conter:

- Sobre
- Áreas
- Processo
- Depoimentos
- FAQ

Separador visual.

Botão:

"Agendar sessão"

Esse botão deve possuir:

- rounded-full;
- shadow-soft;
- largura total;
- destaque visual.

Adicionar animações suaves de entrada.

Inspirar-se em:

- Apple;
- Notion;
- Stripe;
- clínicas premium.

Manter a identidade visual atual do projeto.

Não alterar a navbar desktop.

Mostrar os diffs dos arquivos modificados.
